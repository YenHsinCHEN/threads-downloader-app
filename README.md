# Threads Video Downloader

一個用於下載 Threads 影片的 Android 應用程式

## 功能特點

- ✅ 內建 WebView 瀏覽 Threads
- ✅ 自動偵測頁面中的影片
- ✅ 顯示影片作者和內容預覽
- ✅ 支援批量下載
- ✅ 全選/取消全選功能
- ✅ 自動命名包含作者名稱

## 系統需求

- Android 7.0+ (API 24+)
- 網路連接
- 儲存權限（Android 9 及以下版本）
